How to use
=============

Prerequisites
-------
-  NodeJS
-  NPM
-  Webpack
-  Webpack-dev-server

Installation
-----------
Open the root of the folder in Terminal (Console) and type this commands:

```
npm install
```

Usage
-----------

```
npm start
```

What's inside
-----------
- min Requirements just for running React App with Hot Reloading (no extra files);
